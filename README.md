# timed

Timed is a tiny menu bar timer for macOS.

## Install

Install via Homebrew tap:

```bash
brew tap plainwork/timed https://github.com/plainwork/homebrew-timed
brew install --cask timed
```

Or install directly:

```bash
brew install --cask plainwork/timed/timed
```

## Use

- Click the Timed icon in the menu bar to open the timer.
- Set hours, minutes, and seconds, then press Start.
- Pause/resume as needed, or Cancel to stop.
- Pick an alert sound before starting (No Sound is the default).
- Quit from the app menu or press `command+Q`.
